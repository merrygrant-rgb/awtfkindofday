# A WTF Kind of Day - CMS Publishing Platform

Complete content management system for your humor blog.

## Quick Start

1. Upload to GitHub
2. Connect to Netlify
3. Enable Identity & Git Gateway
4. Start publishing at `/admin`

## Full Instructions

See `COMPLETE_BEGINNERS_GUIDE.md` for step-by-step instructions.

## What's Included

- ✅ Eleventy static site generator
- ✅ Decap CMS for content management
- ✅ Your complete design (preserved)
- ✅ Blog post management
- ✅ Merchandise management
- ✅ Reader submissions
- ✅ Newsletter integration ready
- ✅ Mobile responsive
- ✅ Zero coding required

## Build & Deploy

```bash
npm install
npm run build
```

Deploys automatically via Netlify when you push to GitHub.

## Admin Panel

Access at: `https://yoursite.com/admin`
